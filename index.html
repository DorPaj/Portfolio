Wool War: Roblox Edition

The inspiration for this game was my Minecraft world. I made something like my Minecraft world but on Roblox and I added more stuff to it. In this game, players in their worlds can talk, fight, place blocks and destroy blocks. Users do not have to worry about losing their progress - all worlds are being saved, and leaderboards keep track of what players do. This project shows that:

  • I understand how to script games made of multiple places,
  • I can make reserved servers system and do other things related to TeleportService,
  • I am able to make building system,
  • I understand how to make customization system,
  • I can script game passes and developers products to monetize a game,
  • I am able to script user interface navigation.
  • I understand how to make a normal data store system and a ordered data store system

Here is the actual game but this link can be also found in the post mentioned earlier. This project shows that:
